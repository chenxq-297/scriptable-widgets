# scriptable-widgets

A collection of customizable Apple widgets built with Scriptable.

This repository is used to organize, develop, and share Scriptable-based Apple widgets for iPhone and iPad home screens. It focuses on practical widgets that are easy to customize, maintain, and extend.

## Features

- Built for the Scriptable app
- Customizable widget scripts for Apple devices
- Suitable for personal dashboards and daily information display
- Easy to expand with more widget examples over time

## Goals

- Collect reusable Scriptable widget scripts
- Keep widget code clean and easy to modify
- Provide a simple starting point for new widget ideas

## Getting Started

1. Install the Scriptable app on your iPhone or iPad.
2. Copy a widget script from this repository into Scriptable.
3. Configure the script if needed.
4. Add the Scriptable widget to your home screen and select the script.
